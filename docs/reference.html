run_as_daemon(*aws: Iterable[Awaitable | Task]) → AbstractAsyncContextManager[Task]Â¶

Returns an async context manager that runs all given tasks concurrently with each other and alongside the code inside the with-block, then waits until the code inside the with-block completes. As soon as that happens, the tasks will be cancelled if they are still running.

This is equivalent to trio_util.run_and_cancelling().

async with run_as_daemon(async_fn0(), async_fn1()) as tasks:
    ...
for i, task in enumerate(tasks):
    if task.finished:
        print(f"async_fn{i} completed with a return value of {task.result}.")
    else:
        print(f"async_fn{i} was cancelled.")

Changed in version 0.10.0: Now accepts multiple tasks. The object bound in the as-clause is a list of Task instances instead of a single one.

run_as_main(*aws: Iterable[Awaitable | Task]) → AbstractAsyncContextManager[Task]Â¶

Returns an async context manager that runs all given tasks concurrently with each other and alongside the code inside the with-block, then waits for all tasks to either complete or be cancelled. As soon as all tasks have done so, the code inside the with-block will be cancelled if it is still running.

async with run_as_main(async_fn0(), async_fn1()) as tasks:
    ...
for i, task in enumerate(tasks):
    if task.finished:
        print(f"async_fn{i} completed with a return value of {task.result}.")
    else:
        print(f"async_fn{i} was cancelled.")

If no tasks are given, it simply runs the code inside the with-block.

Changed in version 0.10.0: Now accepts multiple tasks. The object bound in the as-clause is a list of Task instances instead of a single one.

async wait_any(*aws: Iterable[Awaitable | Task]) → Awaitable[Sequence[Task]]Â¶

Runs multiple tasks concurrently, then waits until either one completes or all are cancelled. As soon as one completes, the others will be cancelled.

    if task.finished:
        print(f"async_fn{i} completed with a return value of {task.result}.")
    else:
        print(f"async_fn{i} was cancelled.")
wait_any_cm(*aws: Iterable[Awaitable | Task]) → AbstractAsyncContextManager[Task]Â¶

Returns an async context manager that runs all given tasks concurrently with each other and alongside the code inside the with-block, then waits until either the code inside the with-block completes or any of the tasks completes. As soon as that happens, any remaining tasks and the code inside the with-block (if still running) will be cancelled.

async with wait_any_cm(async_fn0(), async_fn1()) as tasks:
    ...
for i, task in enumerate(tasks):
    if task.finished:
        print(f"async_fn{i} completed with a return value of {task.result}.")
    else:
        print(f"async_fn{i} was cancelled.")
